Overview

What this setup includes

npm scripts

Use npm run dev for local work and npm run build for production output.

Vite build

Fast static bundling with a relative asset base so project Pages deployments work reliably.

GitHub Actions

A deployment workflow publishes the generated dist folder to GitHub Pages on every push to main.

Workflow

From local edit to live page

  1. Edit content and styles in the source files.
  2. Run a local build to verify the site.
  3. Push to GitHub and let Actions deploy automatically.
